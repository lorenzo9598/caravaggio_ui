API prevista

Costruttore base

  • leading, header, content, footer — Widget
  • tags — List<Widget>, popola header se header assente
  • trailing — override; default chevron se onTap != null
  • onTap — nullable, abilita InkWell + freccia
  • Card: sfondo, bordo, ombra · altezza intrinseca

Factory

  • CTile.simple — title, description?, tags (string), icon?, leading?, footer?, onTap?
  • CTile.withDateTime — come simple + DateTime in header (destra)
  • CTile.icon — icon + title + description

Condiviso

  • CTagChip — estratto da CCarousel, usato da entrambi
