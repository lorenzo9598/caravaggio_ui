Costruttore completo (Widget)

  • width obbligatorio
  • aspectRatio default 16/9
  • Uno solo: image Widget · gradient · color
  • backgroundColor — content stacked / scrim overlay
  • title, subtitle?, description?, bottom? — Widget
  • tags — List<Widget>, top-left
  • onTap? — se non null, card cliccabile

Factory semplificati (*Simple)

  • title, subtitle?, description? — String
  • tags — List<String> → chip automatici
  • image — ImageProvider + boxFit
  • CText predefiniti (bianco in overlay)
  • stackedSimple / overlaySimple
